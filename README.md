# NotePad
=========<br>
## NotePad的拓展<br>
  *添加查询功能（根据标题）<br>
  *NoteList中显示条目增加时间戳显示<br>
  *更改记事本的背景<br>
  *UI美化<br>
## 添加的代码<br>
  ### *查询功能：添加了addSearchView函数<br>
  ### *时间戳显示：在NoteEditor中的updateNote函数中获取到当前时间，再添加到values当中以在显示条目当中显示，<br>
  ### *最终显示的是最后修改时间  <br>
  ### *更改记事本的背景：<br>
  ### *添加浮动按钮来设置当前的背景颜色，所使用的背景渐变颜色来自于自己写的XML文件<br>
